# 路笛诗斋 - 安装说明

## 什么是 PWA？

PWA（Progressive Web App）是一种网页应用，可以像原生 App 一样安装到手机上。它的优势：
- 🚀 安装后从桌面启动，全屏运行
- 📴 支持离线浏览已收录的诗词
- 🔄 自动更新，无需应用商店审核
- 💾 数据仅存于本机，隐私安全

---

## 安装方法

### 方法一：通过网页服务器安装（推荐）

#### 1. 部署到静态托管服务

**Vercel（推荐，免费）**
1. 访问 [vercel.com](https://vercel.com)，用 GitHub 登录
2. 点击「New Project」→「Upload」
3. 把整个 `poetry-app` 文件夹拖进去
4. 点击「Deploy」，几秒钟后获得一个 `xxx.vercel.app` 网址
5. 用手机浏览器打开这个网址

**Netlify（同样免费）**
1. 访问 [netlify.com](https://netlify.com)
2. 把 `poetry-app` 文件夹拖到页面上
3. 等待部署完成，获得网址

#### 2. 手机安装

**iOS (Safari)**
1. 打开网址
2. 点击底部分享按钮 ⬆️
3. 选择「添加到主屏幕」
4. 命名为「路笛诗斋」，点击「添加」

**Android (Chrome)**
1. 打开网址
2. 浏览器会自动弹出「添加到主屏幕」提示
3. 或点击右上角菜单 →「安装应用」

### 方法二：本地测试（无需服务器）

在电脑上用 Python 快速启动本地服务器：

```bash
# 进入 poetry-app 目录
cd C:\Users\王小川\.qclaw\workspace\poetry-app

# Python 3
python -m http.server 8080
```

然后用手机浏览器访问 `http://电脑IP:8080`

> 注意：本地 HTTP 无法安装为 App，仅供测试

## API 配置

1. 点击「⚙️ 设置」
2. 填写 DeepSeek API Key（新用户免费额度）
   - 注册地址：[platform.deepseek.com](https://platform.deepseek.com)
3. 点击「测试连接」确认可用
4. 设置自动保存

> 你的 API Key 仅存于手机本地，不会上传到任何服务器

## 常见问题

**Q: 安装后打开是空白页？**
A: 检查是否通过 HTTPS 访问，PWA 要求 HTTPS

**Q: iOS 上状态栏是白色？**
A: 已配置为半透明黑色，如需纯黑可修改 meta 标签

**Q: 如何更新？**
A: 重新部署到托管服务，App 会在下次打开时自动更新

**Q: 数据会丢失吗？**
A: 所有数据存储在手机浏览器的 IndexedDB 中，除非清除浏览器数据才会丢失。建议定期用「备份 JSON」功能导出。
